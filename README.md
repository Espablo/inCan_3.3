# Board inCan v3.3 by Espablo

Description of the module: https://forum.supla.org/viewtopic.php?f=33&t=2188


GPIO settings:

RELAY_1 = GPIO5

RELAY_2 = GPIO13

CONFIG (switch x10) = GPIO4

BUTTON_CONFIG (button) = GPIO0
	
BUTTON_1 = GPIO14

BUTTON_2 = GPIO12

SENSOR_1 = GPIO4

SENSOR_2 = GPIO16

DS18B20 = GPIO2

DHTSENSOR (DHT11, DHT22) = GPIO2

LED_CONFIG = GPIO2
